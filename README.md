# 前言

随着新冠疫情的蔓延，对于学校管理者来说，疫情学生的信息管理变得尤为重要。基于此，我们开发了“基于SSM的疫情学生信息管理系统”。以下是本项目的详细介绍。

## 内容介绍

本项目旨在为学校提供一个便捷、高效的疫情学生信息管理平台。通过此系统，管理员可以轻松地录入、查询、修改和删除学生信息，实时掌握学生健康状况，确保校园疫情的有效防控。系统具有以下特点：

1. 界面简洁，操作方便，易于上手。
2. 数据安全，提供多种权限控制，确保信息安全。
3. 灵活的查询功能，可根据多种条件筛选学生信息。

## 技术介绍

- **语言：** Java
- **使用框架：** Spring、Spring MVC、MyBatis
- **前端技术：** JS、Vue、CSS3
- **开发工具：** IDEA/Eclipse
- **数据库：** MySQL 5.7/8.0
- **数据库管理工具：** phpstudy/Navicat
- **JDK版本：** jdk1.8
- **Maven：** apache-maven 3.8.1-bin
- **前端环境：** Node.Js 12/14/16
